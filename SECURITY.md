# Security Policy

## Supported Versions

We actively support and patch security issues in the following versions of `apibolt`:

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do not disclose it publicly. We want to keep the community safe and resolve any issues promptly.

Please report security vulnerabilities by opening a private draft security advisory on GitHub or by contacting the maintainers directly.

### Process

1. **Acknowledgment:** We will acknowledge receipt of your report within 48 hours.
2. **Investigation:** We will investigate the issue and determine the severity.
3. **Fix:** If confirmed, we will work on a fix and release a patched version as soon as possible.
4. **Advisory:** We will publish a security advisory detailing the vulnerability and the fix to inform users.

Thank you for helping keep `apibolt` secure!
